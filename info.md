# YaMusic v1.0.0

Модуль Яндекс Музыки для Hikka/Heroku юзербота.

## Команды

| Команда | Алиас | Описание |
|---------|-------|----------|
| `.ynow` | `.yn` | Баннер текущего трека (картинка 1920x768) |
| `.ytrack` | `.yt` | Скачать текущий трек как mp3 |
| `.ybio` | `.yb` | Вкл/выкл автобио (каждые 30 сек) |
| `.yguide` | `.yg` | Гайд по получению токена |
| `.ydebug` | — | Отладка: сырые данные Ynison + API |

## Конфигурация (`.cfg YaMusic`)

### Основные
| Параметр | Default | Описание |
|----------|---------|----------|
| `token` | — | OAuth-токен Яндекс Музыки (скрытый) |
| `autobio_text` | `{artist} — {title}` | Шаблон bio при воспроизведении |
| `no_playing_bio` | `""` | Bio когда ничего не играет (пусто = не менять) |

### Шрифты
| Параметр | Default | Описание |
|----------|---------|----------|
| `font_bold` | Onest-Bold (GitHub) | URL шрифта Bold (.ttf) |
| `font_regular` | Onest-Regular (GitHub) | URL шрифта Regular (.ttf) |
| `title_size` | 80 | Размер шрифта названия |
| `artist_size` | 55 | Размер шрифта артиста |
| `time_size` | 36 | Размер шрифта таймеров |

### Цвета
| Параметр | Default | Описание |
|----------|---------|----------|
| `text_color` | `#FFFFFF` | Цвет текста |
| `bar_color` | `#FFFFFF` | Цвет заполненной части прогресс бара |
| `bar_bg_color` | `#A0A0A0` | Цвет фона прогресс бара |

### Визуал баннера
| Параметр | Default | Описание |
|----------|---------|----------|
| `blur_radius` | 14 | Радиус размытия фона |
| `bg_brightness` | 0.3 | Яркость фона (0.0–1.0) |
| `cover_radius` | 35 | Радиус скругления обложки |

## Зависимости

```
aiohttp Pillow
```

## Как работает

- **Ynison WebSocket** — прогресс воспроизведения (progress_ms, duration_ms, paused)
- **HTTP API (track.mipoh.ru)** — метаданные трека (title, artist, cover, download_link)
- **PIL** — генерация баннера: blur-обложка фоном + обложка с rounded corners + текст + progress bar
- **Autobio** — loop каждые 30 сек, обновляет bio текущим треком

## Установка

1. `.loadmod YaMusic.py`
2. `.cfg YaMusic` → вставить OAuth-токен
3. `.ynow` — баннер текущего трека
